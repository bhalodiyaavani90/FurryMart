# IMPORTANT: Clear Browser Cache to See PDF Changes

## The PDF generation code has been updated, but you need to clear your browser cache to see the changes.

## Quick Steps to Clear Cache:

### Method 1: Hard Refresh (Fastest)
1. Open the checkout page in your browser
2. Press **Ctrl + Shift + R** (Windows) or **Cmd + Shift + R** (Mac)
3. This will force reload the page and bypass cache

### Method 2: Clear Cache via DevTools
1. Press **F12** to open Developer Tools
2. Right-click on the refresh button (next to address bar)
3. Select "Empty Cache and Hard Reload"

### Method 3: Clear All Cache
1. Press **Ctrl + Shift + Delete**
2. Select "Cached images and files"
3. Click "Clear data"
4. Reload the page
